# Case Studies

The [`cases/`](https://github.com/Yi5817/Genarris/tree/main/cases) directory in
the repository contains crystal structure prediction (CSP) results for 6 organic
molecules studied in the
[Genarris 3.0 paper](https://doi.org/10.1021/acs.jctc.5c01080).

Each case includes:

- **Experimental structures** – From the
  [Cambridge Structural Database (CSD)](https://www.ccdc.cam.ac.uk/solutions/about-the-csd/)
- **Generated structures** – CIF files of crystal structures generated by Genarris
  and relaxed using MACE-OFF23 and PBE+MBD
- **Analysis data** – Total energies from MACE-OFF23 and PBE+MBD methods
